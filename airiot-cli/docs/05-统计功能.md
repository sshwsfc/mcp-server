## 统计功能

### 设备在线状态统计

**命令：**

```bash
airiot stats-online <tableIds...>
```

**参数：**

- `tableIds`：表 ID 列表，可以指定多个

**返回内容包括：**
- 各表的设备总数
- 在线设备数
- 离线设备数
- 在线率

**示例：**

```bash
# 统计单个表的设备状态
airiot stats-online tbl-001

# 统计多个表的设备状态
airiot stats-online tbl-001 tbl-002 tbl-003
```
