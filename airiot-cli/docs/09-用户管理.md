## 用户管理

### 获取当前用户信息

**命令：**

```bash
airiot user
```

**返回内容包括：**
- 用户 ID
- 用户名
- 昵称
- 邮箱
- 角色
- 权限

**示例：**

```bash
airiot user
airiot user -o json
```

### 获取用户列表

**命令：**

```bash
airiot users [选项]
```

**选项：**

| 选项 | 描述 | 默认值 |
|------|------|--------|
| `--filter <json>` | 过滤条件，JSON 格式 | - |
| `--limit <number>` | 返回数量限制 | 50 |

**filter 参数示例：**

```json
{
  "role": "admin",
  "status": "active"
}
```

**示例：**

```bash
# 查询所有用户
airiot users

# 查询管理员用户
airiot users --filter '{"role": "admin"}'

# 返回前 20 条
airiot users --limit 20
```
