## 属性点查询

### 查询表的属性点定义

**命令：**

```bash
airiot tags <tableId>
```

**参数：**

- `tableId`：表 ID

**返回内容包括：**
- 属性点 ID
- 属性点名称
- 数据类型
- 单位
- 是否可写
- 描述信息

**示例：**

```bash
airiot tags tbl-001
airiot tags tbl-001 -o json
```

### 查询记录的属性点数据

**命令：**

```bash
airiot record-tags <table> <recordId>
```

**参数：**

- `table`：表名称（不是 ID）
- `recordId`：记录 ID

**示例：**

```bash
airiot record-tags devices rec-001
airiot record-tags devices rec-001 -o json
```
